👥 People Management System (Python)

People Management System is a terminal-based Python application that allows you to register, manage, and analyze people. It’s perfect for practicing Python programming, working with JSON data, and creating interactive command-line tools.

🚀 Features

Register people with name, age, and profession

View detailed reports of registered people

Search people by name

Administrator area with password protection

Edit or delete registered people

Data persistence using JSON files

🎮 How It Works

Users can add new people and their details are stored in a JSON file.

You can view reports including total number of people, average age, oldest and youngest person.

Search for a person by name to quickly find their details.

The Administrator area allows advanced operations:

Edit or delete records

View employment status reports

💻 How to Run

Make sure you have Python 3.x installed.

Clone this repository:

git clone (https://github.com/NathnF0/People.Manager.git)

Navigate to the project folder:

cd PeopleManagementSystem

Run the program:

python main.py

Follow the on-screen instructions and enjoy managing your people database!
