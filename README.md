# People Management System (Python)

A simple terminal-based system made in Python to register and manage people.

## Features
- Register people (name, age, profession)
- View reports of registered people
- Search people by name
- Administrator area with password
- Edit or delete registered people
- Data persistence using JSON files

## Technologies
- Python
- JSON file storage

## How to run

1. Install Python
2. Download the project
3. Run the file:

python main.py
